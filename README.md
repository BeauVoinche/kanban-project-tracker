# MERN Full-Stack Application

A full-stack web application built with MongoDB, Express, React, and Node.js. This project demonstrates CRUD functionality, REST API design, client/server architecture, and database-backed application development.

## Tech Stack
- React
- Node.js
- Express
- MongoDB
- JavaScript
- CSS

## Features
- User-facing React client
- RESTful Express API
- MongoDB database integration
- Create, read, update, and delete operations
- Client/server folder structure

## Project Structure
/client   React front-end
/server   Express/Node back-end

## Getting Started

### Server
cd server
npm install
npm start

### Client
cd client
npm install
npm start

## Environment Variables
Create a `.env` file in the server folder:

MONGO_URI=your_mongodb_connection_string
PORT=8000

## What I Learned
- Building a full-stack application from front end to back end
- Connecting React to an Express API
- Structuring routes, controllers, and models
- Working with MongoDB and server-side application logic
