# MERN Project Management Dashboard

A full-stack MERN project management dashboard for creating projects, tracking due dates, and moving work through backlog, in-progress, and completed stages.

## Tech Stack

- MongoDB
- Express
- React
- Node.js
- Axios
- Bootstrap

## Features

- Create new projects with due dates
- View projects by workflow status
- Move projects from backlog to in-progress
- Move projects from in-progress to completed
- Delete completed projects
- Highlight overdue project dates
